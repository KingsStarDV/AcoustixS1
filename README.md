My project AcoustixS1 is a fire extinuguisher which unlike other typical extinguisher uses SOUND WAVES to put out a flame.

Using AcoustixS1 is simple-
Turn on the toggle switch and sweep through the different frequencies using the rotary encoder.
The flame will get extinguish as you sweep through the frequencies.

I made this project as this one is unique.
I also wanted to spend my time productively.
This will also act as a line of safety for anyone who makes it.
